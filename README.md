# Plus-One +1 #

This program will print the timestamp +1 minute/hour/day from NOW. (Also will copy to your system clipboard)

# Install #
```
$ npm install -g plus-one
```

# Usage #
```
// Plus one minute
$ plusone -M

// Plus one hour
$ plusone -H

// Plus one day
$ plusone -D

// Help
$ plusone --help
```
